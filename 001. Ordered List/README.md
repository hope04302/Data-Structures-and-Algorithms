# List

List is one of linear data structure.

- ordered list
- linked list
	- circular linked list
	- doubly linked list

It can be used in circulate polynomials and matrix.

- sparse matrix(희소행렬) -> m * n: 2D array
- dense matrix(밀집행렬) -> {row, column, element} * n: 2D array

** polynomial is simular.
