# Stack and Queue

## Stack

"Stack" is one of linear data structure and LIFO structure.
We can "push" and "pop" elements.

- ordered stack
- linked stack

We use stack structure to make calculator which can process brackets.

- prefix notation
- infix notation
- postfix notation

cf) 문자열 역순 처리도 가능

## Queue

"Queue" is one of linear data strucure and FIFO structure.
We can "push" and "pop" elements.

- ordered queue
	- circular ordered queue
- linked queue

## Degue

"Deque" is a queue which can be pushed or put by elements in both of front and back.

- ordered deque
- linked deque
