# Search

**Search** means literally search something. We can "Hit" or "Miss" to find data.

- internal search

- external search

- comparison search

- non-comparison search

## Sequential Search

**Sequential Search** is easy way to search data.

- basic sequential search

Basic sequential search is also said as "Basic Sequential Search" or "Linear Search".

- index sequential search

	- index table

## Binary Search

**Binary Search** is also said as "Interpolation Search". Divide process makes it works fast.

## Binary Tree Search

**Binary Tree Search** is an useful system to search data.

	- keys of left tree < key < keys of right tree

## Hashing

**Hashing** is one of searching methods which uses "Hash Function" and "Hash Table". 
This technique sometimes generate "Synonyms" and "Collisions".

- 키값 밀도 = (사용) / (남은 칸)
- 적제 밀도 = (사용) / (전체)
